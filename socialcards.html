Setup

  • This javascript plugin requires jQuery, and should be loaded after the initial jQuery library.
  • This design uses a Masonry style, like here - http://w3bits.com/css-masonry/. Examples of the style are available in the source of this demo page.
  • All the images you will need for the card headers are included
  • Add the JavaScript of this page to your site, change all the parameters to fit your needs (see below), and test.

Blog

  • url: This is the URL for the RSS feed
  • This library uses the Google Feed Load API for RSS feeds.
  • entryCount: This is a numerical for how many entries to return
  • iconSrc: This is the URL for the header image icon
  • iconSrcAlt: This is the alt text for the header image icon
